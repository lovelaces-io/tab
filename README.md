# Tab

A mobile-first expense management app that makes submitting out-of-pocket business expenses painless. Built because the existing tools I used as an employee were so clunky I'd rather eat the cost than deal with them.

This is a proof of concept / interactive demo — not a production app. It uses mock data, simulated manager approvals, and runs entirely in the browser with no backend. The goal is to explore what a better expense submission experience could feel like.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- Material Symbols

## Running locally

```bash
npm install
npm run dev
```
